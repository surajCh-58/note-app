# Note App

A Django-based Note App that allows users to register, manage profiles, and create and organize notes.

## Project Overview

This project was developed primarily as a learning and practice project for Django. It focuses on understanding and implementing:

* Function-Based Views (FBV)
* URL routing and URL logic
* User authentication and authorization
* CRUD operations
* Django Forms and ModelForms
* User profile management
* File uploads
* Django messages framework
* Database relationships

## Development Approach

### Backend Development

All Function-Based Views (FBV), URL configurations, business logic, form handling, validation logic, authentication flows, and overall backend implementation were written manually by the developer without AI-generated code.

AI tools were used only as a debugging and learning assistant when unexpected errors or issues occurred during manual development. The core backend architecture, logic, and implementation decisions were created and coded independently.

### Frontend Development

The frontend design, Tailwind CSS styling, UI enhancements, and visual modifications were created with assistance from Claude AI. AI was primarily used to improve the appearance, responsiveness, and user experience of the application.
